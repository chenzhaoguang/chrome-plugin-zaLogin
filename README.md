# chrome-plugin-zaLogin
这是一个chrome插件
# 如何使用
```
1、控制台输入
git clone https://github.com/chenzhaoguang/chrome-plugin-zaLogin.git

2、chrome浏览器中输入
chrome://extensions/

3、将chrome-plugin-zaLogin拖入扩展程序即可

```
